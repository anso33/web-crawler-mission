# 나만의 웹 크롤러 만들기

원하는 영화사의 박스오피스 순위를 가져와서 엑셀 or PDF로 만들어주는 프로그램입니다.

## 기능 설명

1. 원하는 영화사를 선택하면 해당 영화사의 박스오피스 순위 정보를 파일로 생성할 수 있다.
2. 원하는 영화의 상세 정보와 상영 시간표를 파일로 생성할 수 있다.
